DM file downloads

You will be able to download results in this tab after you run Dynamic Model.

  • Follow the instructions in the "Help" tab for uploading data and running models.
  • Once the Dynamic Model run has completed, in this tab you'll be able to download the following results files:
    • PDF summary of Dynamic Model results
    • Rav file for input to VRAP
    • RData file containing the model posteriors. This can be used as an input to DM for regeneration of text and graphical results.

Note that if you change the model parameters or the input type or upload a new input file, any previously generated outputs will no longer be available for download.
