Dynamic Model and VRAP: Running DM

For information on running VRAP, select the "VRAP" tab and click "Help".

From the DM tab, you can

  • Run Dynamic Model on uploaded XLSX files, previously generated DM RData files, or a demo A&P file.
  • Examine and download DM results.

Running DM

  • Configuration
    • Select an input type, either an A&P XLSX file, an RData file saved from a previous DM run, or the demo A&P file.
    • For XLSX and demo files, select a spawner-recruitment (SR) function : either Ricker, Beverton-Holt, or Hockey Stick.
    • Specify whether Marine Survival and Stream Flow covariate effects should be included.
    • Select the analysis type, either DM (traditional) or SS (state space model that includes age data).
    • In the "Priors" tab, adjust the settings of priors parameters.
    • Click the "Run Dynamic Model" button.
  • Results
    • Once DM has run, examine results in the "DM A&P Data" and "DM Posteriors" sub-tabs.
    • From the "Downloads" sub-tab, download a summary report, an output RAV file, and an RData file containing R objects generated by the run.
    • If any input settings are altered following a run, output files will no longer be available for viewing or downloading. Download any desired files before altering settings.
