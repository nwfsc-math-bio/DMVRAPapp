File downloads

You will be able to download results in this tab after you run the Dynamic Model or the Dynamic Model followed by VRAP.
  • Follow the instructions in the info tab for uploading data and running models.
  • If you ran only the Dynamic Model, look in this tab to download these results files:
    • PDF summary of Dynamic Model results
    • .rav file to run independently in VRAP
    • RData file containing the model posteriors
  • If you ran the Dynamic Model and VRAP, you can download the DM results file described above as well as VRAP results:
    • Summary data
    • Brood year
    • Escapement

Note that if you change the model parameters or the input type or upload a new input file, any previously generated outputs will no longer be available for download.
