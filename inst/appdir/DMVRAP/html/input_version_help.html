Input file versions

The new version of the population dynamics model allows age data to be directly integrated. This is in contrast to the original version of the model where age data was used indirectly through the maturation rates calculated in the age engine. Therefore, the the DynamicsInput sheet in the A&P workbook used for the new model needs to include raw age data. This is accomplished by linking or copying the age data in columns C through T in the inputAge sheet into columns AE through AP in the dynamicsInput sheet. Note that the columns in the DynamicsInput sheet only include the age data and not sums. So there should be 4x3=12 columns total for freshwater age 0 origin, freshwater age 1 origin, and the total (ages 2,3,4,5).

For example, to modify an existing DynamicsInput sheet to work with the new package model, additional columns can be generated by linking to the appropriate InputAge data. The resulting DynamicsInput column contents might look like this:

Column Content
AE13=IF($InputAge.C8="","",$InputAge.C8)
AF13=IF($InputAge.D8="","",$InputAge.D8)
AG13=IF($InputAge.E8="","",$InputAge.E8)
AH13=IF($InputAge.F8="","",$InputAge.F8)
 
AI13=IF($InputAge.I8="","",TRUNC($InputAge.I8))
AJ13=IF($InputAge.J8="","",TRUNC($InputAge.J8))
AK13=IF($InputAge.K8="","",TRUNC($InputAge.K8))
AL13=IF($InputAge.L8="","",TRUNC($InputAge.L8))
 
AM13=IF(AE13="","",AE13+AI13)
AN13=IF(AF13="","",AF13+AJ13)
AO13=IF(AG13="","",AG13+AK13)
AP13=IF(AH13="","",AH13+AL13)

with the formulae being replicated from row 13 down to the final data row.

(Formula syntax like "$InputAge.C8" may be replaced by "InputAge!C8" or other forms, depending on the spreadsheet software used.)
