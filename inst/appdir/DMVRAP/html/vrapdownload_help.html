VRAP file downloads

You will be able to download results in this tab after you run VRAP.

  • Follow the instructions in the "Help" tab for uploading data and running a model.
  • Once the VRAP run is complete, you'll be able to download the following results files:
    • Summary information
    • Brood year results
    • Escapement results

Note that if you change the model parameters or the input type or upload a new input file, any previously generated outputs will no longer be available for download.
