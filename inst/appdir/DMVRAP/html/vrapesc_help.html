VRAP Escapement Results

Escapement results from VRAP are displayed in this tab when VRAP has been run.

To run VRAP:

  • Follow the instructions in the info tab for uploading data and running models.
  • Run the Dynamic Model and VRAP.
  • Once the Dynamic Model has been computed and VRAP has run, VRAP summary results will appear in this tab.

Note that if you change the model parameters or the input type or upload a new input file, any previously generated results will no longer be displayed.
