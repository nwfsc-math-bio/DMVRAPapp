Dynamic Model and VRAP : Running VRAP

For information on running Dynamic Model, select the "DM" tab and click "Help".

From the VRAP tab, you can

  • Select an input type: either a RAV file, a demo file, or the most recent DM output (if one exists).
  • Examine and download VRAP results.

Running VRAP

  • Configuration
    • Select an input type: either a RAV file, a demo file, or the current DM results. (DM results - and this option - are only available if DM has been run and no configuration changes have been made in the DM tab.)
      - If you select the demo file input type, you'll also be able to download copies of the demo files.
    • Select the desired number of runs per simulation.
    • Click the "Run VRAP" button.
  • Results
    • Once VRAP has run, examine the results in the "VRAP Summary", "VRAP Brood Year", and "VRAP Escapement" sub-tabs.
    • In the "Downloads" sub-tab you can download copies of the VRAP results summary, the broody year results, and the escapement results.
    • If any input settings are altered following a run, output files will no longer be available for viewing or downloading. If VRAP was run on current DM results and DM settings are changed, VRAP results will not be available. Download any desired files before altering the app settings.
