Dynamic Model and VRAP : Running VRAP

For information on running Dynamic Model, select the "DM" tab and click "Help".

From the VRAP tab, you can

  • Select an input type: either a RAV file, a demo file, or the output of the most recent DM run.
  • Examine and download VRAP results.

Running VRAP

  • Configuration
    • Select an input type: either a RAV file, a demo file, or the current DM results. DM results - and the option to use them as inputs to VRAP - are only available if DM has been run and no configuration changes have been made in the DM tab since the run.
      - If you select the demo file input type, you'll also be able to download copies of the demo files.
    • Select the desired number of runs per simulation.
    • Click the "Run VRAP" button.
  • Results
    • Once VRAP has run, examine the results in the "VRAP Summary", "VRAP Brood Year", and "VRAP Escapement" sub-tabs.
    • In the "Downloads" sub-tab you can download the VRAP results summary, the brood year results, and the escapement results. For some model types, a PDF summary report will also be available.
    • If any input settings are altered following a run, output files will no longer be available for viewing or downloading. If VRAP was run on current DM results and DM settings are changed, VRAP results will not be available. Download any desired files before altering settings.
