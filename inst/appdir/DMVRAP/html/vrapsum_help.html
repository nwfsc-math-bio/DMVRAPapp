VRAP Summary Results

Summary results from VRAP are displayed in this tab when VRAP has been run.

To run VRAP:

  • Follow the instructions in the "Help" tab for uploading data and running models.
  • Run Dynamic Model and VRAP or VRAP on its own.

Note that if you change the model parameters or the input type or upload a new input file, any previously generated results will no longer be displayed.
